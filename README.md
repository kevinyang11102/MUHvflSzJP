# Python大学生就业信息管理系统 python74

## 项目概述

本项目为基于Python语言的B/S架构的大学生就业信息管理系统。系统旨在为大学生提供便捷的职位查询、投递管理等功能，同时为企业及管理员提供一个高效的信息发布与管理的平台。

## 技术栈

- **开发工具**：IDEA、Eclipse
- **编程语言**：Python
- **数据库**：MySQL 5.7+
- **框架**：Django

## 功能模块

- **首页**：展示系统概览及最新职位信息。
- **职位管理**：包含职位发布、编辑、删除等功能。
- **职位列表**：显示所有职位信息，支持筛选与排序。
- **企业管理**：对企业信息进行管理。
- **留言管理**：管理用户留言，提供互动平台。
- **权限认证**：用户权限管理，确保系统安全。
- **用户管理**：对系统用户进行管理。
- **投递管理**：管理用户的职位投递记录。

## 系统角色

- **普通用户**：查询职位、投递简历等。
- **企业用户**：发布职位、管理职位、查看简历等。
- **管理员**：系统维护、用户管理、职位审核等。

## 运行环境

- Python环境搭建（建议使用虚拟环境）。
- MySQL数据库服务。
- 支持Python的服务器。

## 部署步骤

1. 配置Python环境及MySQL数据库。
2. 导入数据库文件。
3. 部署至服务器，配置相关服务器参数。
4. 运行系统。

## 目录结构

```
|-- 项目名称
    |-- apps               # 应用目录
    |-- migrations         # 数据库迁移文件目录
    |-- static             # 静态文件目录
    |-- templates          # 模板文件目录
    |-- utils              # 工具目录
    |-- manage.py          # 管理脚本
    |-- requirements.txt   # 依赖列表
```
